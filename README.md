# Learn Platform

Seluruh kode dan konfigurasi proyek ada di folder **`apps/`**.

```bash
cd apps
make help          # daftar perintah
make setup         # install dependencies
make db-setup-local
make backend-dev   # API :8080
make frontend-dev  # UI  :3000
```
